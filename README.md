# 后台权限管理系统 java876

## 项目概述

本项目为后台权限管理系统，提供两种技术栈版本供选择。

## 技术栈

### 版本一

- 后端：SpringBoot
- 安全框架：Shiro
- 数据库：MySQL
- 前端：BootStrap

### 版本二

- 后端：SpringBoot
- 安全框架：SpringSecurity
- 缓存：Redis
- 数据库：MySQL
- 前端：Vue

## 功能模块

- 用户管理
- 角色管理
- 权限控制
- 资源管理
- 操作日志

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- JDK 1.8 或以上版本
- MySQL 5.7 或以上版本
- Redis（版本二需要）

## 部署步骤

1. 准备运行环境
2. 创建数据库
3. 导入初始数据
4. 编译项目
5. 部署项目
